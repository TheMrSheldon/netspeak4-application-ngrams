178 // We know that all generated concats have the same min_length (because
179 // min_length doesn't depend on the order of the elements in the concat), so
180 // if a concat of all elements of the order set has a min_length >
181 // options.max_length, then we know that all norm queries generated from
182 // this order set will be discarded anyway. Thus we don't need to do any
183 // work and can just return an empty alternation.
184 //
