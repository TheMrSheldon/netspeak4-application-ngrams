uint32_t netspeak::regex::RegexQuery::combinations_upper_bound ( ) const

Returns an upper bound for the number of possible words, the query can match. If UINT32_MAX is returned, the upper bound is essentially infinite. If 0 is returned, the query cannot match any word.

Queries which contain QMARK or star units will automatically return UINT32_MAX .
