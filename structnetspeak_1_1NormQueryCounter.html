std::function< uint64_t(size_t)> netspeak::NormQueryCounter::estimate ( const SimpleQuery &  query)
inline

Returns a function that, given the number of words each regex will be replaced with, return an estimate for the number of norm queries creates by the given query.

The returned estimate is guaranteed to be always higher than the actual number of norm queries (but also bounded by the maximum number representable in uint64_t ).

Every regex in the given query will be assumed to be replaced by the given number of words.
