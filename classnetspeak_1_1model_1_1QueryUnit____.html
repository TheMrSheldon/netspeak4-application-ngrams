◆ text()

const std::string & netspeak::model::QueryUnit__::text ( ) const
inline

The unescaped text value of the unit.

"Unescaped" means that backslashes used to escape characters (e.g. in words and regexes) and prefixes used to identify the token (e.g. the # of dict sets) are already handled.

This value is only relevant for terminals. The text of non-terminals is empty.
