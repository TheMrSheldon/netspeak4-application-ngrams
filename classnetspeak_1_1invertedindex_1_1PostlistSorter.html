Detailed Description

template<typename T, typename Comp>
class netspeak::invertedindex::PostlistSorter< T, Comp >

A class that provides a static method to sort a huge postlist with an external n-merge-sort algorithm. If the size of the postlist is less than max_memory_usage_mb sorting falls back to the std::sort algorithm.
