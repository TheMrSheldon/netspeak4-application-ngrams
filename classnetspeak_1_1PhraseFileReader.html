Detailed Description

template<bool stream_provides_phrase_id>
class netspeak::PhraseFileReader< stream_provides_phrase_id >

Parses text files in the ngram-file format, which defines a line as follows: <ngram> TAB <frequency>

Optionally there can be an additional value per line to assign some phrase id (to set as template parameter). The line format then reads as follows: <ngram> TAB <frequency> TAB <id>
