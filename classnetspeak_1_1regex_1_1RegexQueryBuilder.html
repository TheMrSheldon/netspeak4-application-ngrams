void netspeak::regex::RegexQueryBuilder::add ( RegexUnit  unit)

Adds the given unit to the internal query.

The builder might or might not actually add the given unit to the query. The actual query is considered an implementation detail, so any number of units might be added, changed, or removed as a result of this operation.

The builder only guarantees is that the resulting query will behave as if the given unit had been added.
