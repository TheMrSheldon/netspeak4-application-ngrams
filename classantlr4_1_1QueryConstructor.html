Detailed Description

A query listener that will construct a Query while visiting all ANTLR4 grammar rules.

The understand the implementation of this class, it's helpful to have the grammar (not necessarily the lexer) open on the side.

Todo:
This class smells fishy. The stack should be replaceable with a Visitor implementation. Right now, it is a stack-machine on top of the stack-machine of the parser.
