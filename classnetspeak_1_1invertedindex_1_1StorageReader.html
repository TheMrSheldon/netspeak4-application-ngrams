inline

Returns the postlist for a given key, which may be truncated at the the beginning by the value of begin and at the end by the value of length. If a mapping for key exists but the other parameters truncate the postlist completely, then an empty postlist will be returned. Otherwise, if no mapping exists at all NULL will be returned to indicate that case.

Parameters
keythe key of the wanted postlist.
beginthe index of the first element.
lengththe number of the element to read.
page_sizethe size of the postlists swap buffer in bytes.
Returns
a postlist, which can be empty or null.
