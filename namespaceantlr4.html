static void antlr4::unescape_word ( std::string &  str)
static

Unescapes backslashes in the given string.

Unescapes backslashes in the given string. That is, sequences of the form \c, were cis an arbitrary character, are replaced with c in the output. In effect this means that single backslashes are removed and double backslashes are replaced with a single backslash:

>>> unescape_word("hello world");
"hello world"
>>> unescape_word("\\hello \\\\world\\");
"hello \\world"
>>> unescape_word("hello \\\\\\world");
"hello \\world"
